## How to Take Screenshots

1. Start the dev server: `npm run dev`
2. Open http://localhost:3000
3. Navigate through each screen listed above
4. Take screenshots (use Cmd+Shift+4 on Mac or Snipping Tool on Windows)
5. Save with the exact filenames listed above
6. Place all screenshots in this directory
